# Pygame Info
Pygame should be installed for both projects with: <br>
	```py -m pip install -U pygame --user``` <br>

After that both projects can be run with: <br>
	``py Full_Game.py`` <br>
	``py main.py`` <br>
	
> [!Note]
> py is standing for the python equivalent used on your device, for example: python.exe <br>	
> Please note that the audio output for the BCX project only works on Windows. <br>
